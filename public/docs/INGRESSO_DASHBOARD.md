# Guia do Usuário — Dashboard de Validação da Documentação

Este guia explica as funcionalidades da página "Validação da Documentação" e mostra como realizar as operações mais comuns de forma prática.

Principais objetivos:
- Visualizar o progresso da validação de documentos por convocação.
- Acessar rapidamente listas filtradas e realizar encaminhamentos.

---

## Onde começar

- Título da página: **VALIDAÇÃO DA DOCUMENTAÇÃO** — indica o contexto atual.
- Botões principais (no canto superior direito):
	- **Ver todos os convocados** — abre a lista completa filtrada pela convocação ativa.
	- **Aptos para encaminhamento** — abre a lista de candidatos aptos para encaminhamento.
	- **Guia da Página** — abre este manual em um modal para referência rápida.

---

## Abas de Convocação

- Cada aba representa uma convocação (ex.: 1ª Convocação, 2ª Convocação).
- Ao clicar para mudar de aba, você verá uma confirmação. A mudança só é aplicada se você confirmar.
- Depois de confirmar, o sistema mantém a convocação ativa e sincroniza o link `Ver todos os convocados` para abrir a lista correta.

Dica rápida: confirme a aba correta antes de abrir a lista completa para evitar filtragens inesperadas.

---

## Painéis e Métricas

- **Cartões de Métricas** — mostram contagens por convocação:
	- Total de Candidatos
	- Corrigir documentação (exige correção)
	- Pendência de Docs
	- Docs Validados
	- Pendente Confirmação CPM

- **Painel COP** (apenas para usuários com permissão): mostra número do COP, quantidade total, candidatos atribuídos e COPs livres.

- **Gráfico por NTE** — visualiza a distribuição de candidatos por núcleo/entidade (use para priorizar encaminhamentos).

---

## Resumo e Ações Rápidas

- No painel lateral (Resumo) você encontra badges com números rápidos:
	- Candidatos totais
	- Pendências
	- Corrigir documentação
	- Não Assumiu
	- Aptos para encaminhamento

- Use os botões para abrir as listas filtradas e tomar ações como validação de documentos ou encaminhamento.

---

## Passos Comuns (Exemplos)

- Ver lista completa da convocação atual:
	1. Selecione a aba da convocação.
	2. Confirme a troca quando solicitado.
	3. Clique em **Ver todos os convocados**.

- Encaminhar candidatos aptos:
	1. Clique em **Aptos para encaminhamento**.
	2. Abra a lista e selecione os candidatos para encaminhar.

---

## Resolução de Problemas (Rápido)

- Não aparece confirmação ao trocar de aba: atualize a página e garanta que você não tem múltiplas versões do Bootstrap/jQuery carregadas.
- Números incorretos no resumo: atualize a página e verifique se a convocação correta está ativa.

---

## Perguntas Frequentes

- Posso trocar a convocação sem confirmar? — Não. O sistema exige confirmação para evitar mudanças acidentais.
- O que significa "Aptos para encaminhamento"? — São candidatos cuja documentação foi validada e que estão prontos para serem encaminhados para a unidade.
